TypeScript 入门指南

TypeScript 是 JavaScript 的超集，添加了类型系统。

为什么使用 TypeScript

  1. 更好的类型检查
  2. 更好的 IDE 支持
  3. 更易于维护
interface User {
  name: string;
  age: number;
}

function greet(user: User): string {
  return \`Hello, \${user.name}!\`;
}
